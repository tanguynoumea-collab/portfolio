## Context

A quarterly cultural magazine published in French and English wanted to unify its layout. Each issue was composed by hand, which made the design unpredictable and time-consuming. I was tasked with designing a robust editorial grid able to host long articles, photo portfolios, and interviews within the same visual coherence.

## Challenge

The difficulty lay in the bilingualism: the same article takes roughly 15% more space in French than in English. The grid had to absorb that variation without breaking the vertical rhythm or forcing the layout artist to reinvent every spread. It needed to be a flexible yet constraining system, legible both on screen and in print.

## Process

I built a twelve-column grid paired with a baseline grid set to the body text leading. Every element — headings, captions, pull quotes — aligns to that framework, which guarantees a constant vertical rhythm from page to page.

<Callout variant="info" title="Vertical rhythm">
  A baseline grid shared by all paragraph styles keeps adjacent columns aligned even when text length varies from one language to another.
</Callout>

I then derived a three-level typographic hierarchy and defined InDesign templates for each article type. Layout artists now start from a template rather than a blank page, and length differences between language versions are handled through breathing zones planned in advance.

## Outcome

The magazine now composes a full issue in half the time it used to take, the grid natively absorbing the gap between the two languages. Visual consistency from issue to issue improved markedly, and reader feedback on legibility was positive. The template system is now used by the entire editorial team.
